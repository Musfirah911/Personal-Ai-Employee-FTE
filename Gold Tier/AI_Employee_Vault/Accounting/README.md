# Accounting — Odoo Integration

This folder stores accounting snapshots pulled from Odoo Community via the MCP server.

## Files
- `latest_snapshot.md` — last pulled P&L + invoice summary (auto-generated)
- `audit_YYYY-MM-DD.md` — weekly audit reports (generated by accounting-audit skill)

## How it works
1. `odoo_watcher.py` polls Odoo JSON-RPC every 15 min
2. Writes a trigger to `/Needs_Action/ODOO_sync_<timestamp>.md`
3. Orchestrator runs the `accounting-audit` skill
4. Skill queries Odoo MCP → writes snapshot here
5. Weekly briefing skill reads this folder for CEO report

## Odoo Setup
- Install: Odoo Community 17+ (local Docker recommended)
- MCP Server: https://github.com/AlanOgic/mcp-odoo-adv
- Config: Set ODOO_URL, ODOO_DB, ODOO_USER, ODOO_PASSWORD in .env
